Salferrarello.com Mods
======================

Since I'm running an unmodified version of
[Bootstrap Genesis](http://ironcodestudio.com/bootstrap-genesis-theme/)
on [salferrarello.com](salferrarello.com), I'm placing my customizations
in this plugin.
